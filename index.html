The Movie Recommendation System is built using machine learning techniques to suggest relevant movies based on a selected title. It applies content-based filtering by extracting features like genres and descriptions, then uses TF-IDF vectorization to convert text into numerical format. Cosine similarity is calculated to measure the closeness between movies, allowing the system to recommend top matches. The model is trained and evaluated using Python libraries like Scikit-learn and Pandas. Key ML concepts applied include text processing, vectorization, feature extraction, and similarity scoring, making it a practical application of machine learning in recommendation systems.

The Weather Forecasting System is a machine learning‑powered tool designed to forecast key weather parameters—such as temperature, humidity, and rainfall—using historical Indian meteorological data. The project begins with extensive data preprocessing, addressing missing values, feature engineering (like lag variables and seasonal categorization), and normalization. It employs regression algorithms—including Linear Regression and Random Forest Regressor—to train predictive models with cross‑validation, optimizing for accuracy and robustness. Visualization tools like Matplotlib and Seaborn are used to illustrate weather trends, error distributions, and feature importances. The system is built with Python and deployed as an interactive app via Streamlit, enabling users to enter a location and date to receive real‑time, location‑based forecasts with intuitive graphs and insights.
